# 🚀 Elite Residences / PropPulse AI
### *The Next-Gen AI Real Estate Agent for Ludhiana*

**Built by Adarsh | 10th Grade Student | Mobile-First Developer**

---

## 🌟 The Vision
Building a "Unicorn" company starts with solving local problems. **PropPulse AI** is an autonomous agent designed to navigate the complex real estate market of Punjab. It uses advanced LLM logic to match residents with their ideal homes, bypassing traditional friction.

## 📱 The "Hard Mode" Challenge
This entire project was **architected, coded, and deployed using only a mobile device.** * **Constraint:** 0% Budget, 100% Sweat Equity.
* **Daily Grind:** Developed in 1.5-hour "sprints" between 12-hour JEE preparation sessions.
* **Tech Stack:** HTML5, CSS3, JavaScript, Gemini 1.5 Flash API, and Vercel.

## 🛠️ Features
- **Smart Matching:** AI-driven property recommendations based on natural language input.
- **Ludhiana-Centric Logic:** Specialized knowledge of local areas like Sarabha Nagar, Model Town, and Ram Nagar.
- **Zero-Latency Deployment:** Optimized for mobile-web performance.
